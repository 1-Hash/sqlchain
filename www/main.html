Overview sqlChain is a compact SQL layer that runs on top of bitcoind. It enhances the ability to query the blockchain with a priority placed on low storage and processing overhead. This demonstration server provides an Insight (mostly) compatible API. The API page above documents the queries supported and observed differences to the standard API, as well as other technical details.

Using sqlChain over a pruning node, as a compact alternative to a full node, and more server applications based on sqlChain are currently being explored and under development.
